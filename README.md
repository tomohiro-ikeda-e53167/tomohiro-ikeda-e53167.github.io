# UTM座標抽出 for CoordyNavi

## 1. 概要
- ブラウザの **GPS（Geolocation API）** を使って現在地（緯度・経度）を継続取得します（追跡）。
- 取得した緯度・経度を **UTM座標（Zone / Easting / Northing）** に変換します。
- ボタンで「今の座標」を記録（ログに追加）できます。
- 記録したデータを **CSVとして保存** できます。

CoordyNaviの補助用途として、**今いる地点を何点か記録して後でCSVで回収します。**
保存したCSVをCoordyNaviで使用します。

## 3. 中でやっていること
- 緯度経度→UTM変換
  - 入力：緯度、経度
  - 出力：`{ zone, easting, northing }`
- GPS追跡
  - 継続的に位置更新します
- 現在の地点を記録
- CSV保存
- 自動開始
  - ページを開いたら自動でGPS追跡を開始します。
  - 位置情報取得の許可が必要です。

---

## 4. 使用方法
1. [ここ](https://tomohiro-ikeda-e53167.github.io/index.html)をブラウザで開きます
2. ブラウザの位置情報許可ダイアログが出たら **許可** する
3. 画面上部に `現在の精度: ±xxm` が表示され、緯度・経度が更新されるのを待つ
4. 記録したい地点で **「今の座標を記録！」** を押す  
5. 記録が終わったら **「CSV保存」** を押して `survey_data.csv` をダウンロードする
---

## 5. 注意点
- 精度（±m）は環境で大きく変わります（屋内、ビル陰、樹木、端末性能など）。
- 端末のバッテリー消費が増えることがあります。
- UTMのゾーンは経度によって変わるため、広域を移動するとゾーンが変わる可能性があります
